# 🧠 Jogo da Memória

Projeto desenvolvido em C# utilizando Windows Forms, com o objetivo de praticar lógica de programação, manipulação de eventos e construção de interfaces gráficas.

## 📷 Demonstração

<img width="794" height="476" alt="image" src="https://github.com/user-attachments/assets/2f1397b9-c2e1-48e4-ba62-c6931d91b4e2" />


## 🚀 Funcionalidades

- Embaralhamento automático das cartas
- Sistema de virar cartas ao clicar
- Verificação de pares
- Contagem de tentativas
- Reinício do jogo

## 🛠️ Tecnologias utilizadas

- C#
- .NET Framework
- Windows Forms
- Visual Studio 2022

## 🎮 Como jogar

1. Execute o projeto
2. Clique nas cartas para virá-las
3. Encontre os pares iguais
4. Complete o jogo com o menor número de tentativas possível

## 📂 Estrutura do projeto

- `Form1.cs` → Lógica principal do jogo
- `Program.cs` → Inicialização da aplicação
- `Resources` → Imagens das cartas

## 💡 Aprendizados

Durante o desenvolvimento deste projeto, foram aplicados conceitos como:

- Manipulação de eventos (Click)
- Uso de arrays/listas
- Lógica de comparação
- Organização de código
